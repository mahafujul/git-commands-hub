ðŸŸ¢ Easy Commands (Everyday Basics)

Repository Setup

git initCreate a new Git repository in current folder
git clone <url>Download a repository from GitHub to your computer
git statusShow what files have changed (your best friend!)
git add <file>Stage a file for commit (prepare it)
git add .Stage all changed files
git commit -m "message"Save changes with a description
git pushUpload your commits to GitHub
git pullDownload latest changes from GitHub

Basic Information

git logSee history of commits
git log --onelineSee commit history in compact format
git diffShow what changes you made
git branchList all branches
git branch <name>Create new branch
git checkout <branch>Switch to another branch
git merge <branch>Combine another branch into current one

ðŸŸ¡ Medium Commands (Getting Comfortable)

Branch Management

git checkout -b <branch>Create and switch to new branch in one command
git branch -d <branch>Delete a branch (safe deletion)
git branch -D <branch>Force delete a branch
git push -u origin <branch>Push new branch to GitHub and track it
git push origin --delete <branch>Delete branch from GitHub

Undoing Changes

git checkout -- <file>Discard changes in a file (back to last commit)
git reset HEAD <file>Unstage a file (opposite of git add)
git commit --amendChange the last commit message or add more files
git revert <commit-id>Create new commit that undoes another commit
git clean -fDelete untracked files permanently

Cherry Picking & Stashing

git cherry-pick <commit-id>Copy a specific commit to current branch
git stashTemporarily save changes without committing
git stash -uStash changes including untracked files
git stash popBring back stashed changes
git stash listSee all your stashes
git stash dropDelete a stash

Remote Management

git remote -vSee which GitHub repository you're connected to
git remote add origin <url>Connect your folder to a GitHub repository
git fetchDownload changes but don't merge them yet
git push --set-upstream origin <branch>Push and track new branch

ðŸ”´ Hard Commands (Advanced Wizardry)

History Rewriting (Dangerous!)

git reset --soft HEAD~1Undo last commit, keep changes staged
git reset --mixed HEAD~1Undo last commit, unstage changes
git reset --hard HEAD~1Undo last commit, delete all changes
git reset --hard <commit-id>Jump back to specific commit, lose everything after
git rebase -i HEAD~3Interactively edit last 3 commits (squash, reorder, etc.)
git rebase <branch>Move your commits to tip of another branch

Advanced History & Recovery

git reflogSee everything you've done (even deleted commits!)
git fsck --lost-foundFind orphaned commits
git log --graph --oneline --allVisual tree of all branches
git blame <file>See who wrote each line of code
git bisect startBinary search to find which commit broke something

Advanced Merging

git merge --no-ff <branch>Force create merge commit (even if fast-forward possible)
git merge --squash <branch>Combine all commits from branch into one
git merge -X ours <branch>Merge but prefer our version in conflicts

ðŸ™ GitHub Concepts

Repository Management

Fork - Make your own copy of someone else's repository
Clone - Download a repository to your computer
Star - Bookmark a repository (like a "like" button)
Watch - Get notifications when repository changes
Issues - Bug reports or feature requests
Pull Request (PR) - Ask to merge your changes into main repository

Collaboration Workflow

Contributing to Open Source

# On GitHub: Fork the repository
git clone <your-fork-url>
git checkout -b feature-branch
# Make changes
git add .
git commit -m "Add amazing feature"
git push origin feature-branch
# On GitHub: Create Pull Request
